# polimino
<h1>General</h1>
<p> Данная програма(скрипт?) работает при следующих условиях:
  <ul>
    <li>Стороны стола являются четными числами</li>
    <li>Правильно считает, если нет фигур типа (x+n,x), где n - натуральное число</li>
  </ul>
